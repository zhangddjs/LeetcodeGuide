## 题目

Your are given an array of positive integers `nums`.

Count and print the number of (contiguous) subarrays where the product of all the elements in the subarray is less than `k`.

## 简述

**输入：** 整型数组; 整型k

**输出：** 乘积小于k的组合数

**Notes：**

+ 0 < 数组长度 <= 50000
+ 0 < 数组元素 <= 1000
+ 0 <= k < $10^6$
