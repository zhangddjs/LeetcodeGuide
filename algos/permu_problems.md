# Perm

| 枚举对象      | 数量级   | 复杂度类型  | 示例              |
| ---------     | -------- | ----------- | ------            |
| 数组元素      | n        | O(n)        | 线性              |
| 所有对 (i,j)  | n²       | O(n²)       | 多项式            |
| 所有三元组    | n³       | O(n³)       | 多项式            |
| k个元素的组合 | C(n,k)   | O(n^k)      | 多项式（k固定时） |
| 所有子集      | 2^n      | O(2^n)      | 指数级            |
| 所有排列      | n!       | O(n!)       | 阶乘级            |
| 所有分区      | Bell(n)  | O(n^n)      | 超指数            |
| 所有树        | n^(n-2)  | O(n^(n-2))  | 指数级            |
