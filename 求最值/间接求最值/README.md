## 涉及到的数学知识

### 三角形面积计算

1.输入3顶点：

`return |(p1.x - p0.x) * (p2.y - p0.y) - (p2.x - p0.x) * (p1.y - p0.y)| / 2`

2.输入3边长：

``` c
s = (a + b + c) / 2;
return sqrt(s * (s - a) * (s - b) * (s - c));
```

3.一个角两个邻边：

`return 0.5 * a * b * sin(C)`
